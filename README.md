# WeatherBot
Проект помогает людям определять возможную вероятность актированного дня(отмены учебных занятий) на следующий день и высылает данную информацию в группу Вк, Телеграмм канал.
Проект представляет из себя веб-сервиса на .NetCore 3.1 с использованием фреймворков AspNetCore, CronScheduler. 
Краткая суть работы заключается в следующем:
С помощью api погоды Яндекса получаем информацию о погоде на следующий день, обрабатываем логикой чтобы получить вероятность актированного дня, далее с использованием api Вк и Телеграма выкладываем информацию о вероятности актированного дня и текущую погоду в соответствующие канал и группу. В течение дня можно настроить любое количество таких постов, в любое время. Так как в городе Воркуте имеется официальная группа ГО и ЧС, которая информирует жителей в каждый день в 6 утра о наличии актированного дня, то бот при выкладывании фактической информации в 6 утра берет информацию об актированном дне с данной группы и выкладывает информацию в свою группу и телеграм канал.
