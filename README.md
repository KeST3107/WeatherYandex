# WeatherBot
Проект помогает людям определять возможную вероятность актированного дня(отмены учебных занятий) на следующий день и высылает данную информацию в группу Вк, Телеграмм канал.  
Проект представляет из себя веб-сервис на .NetCore 3.1 с использованием фреймворков AspNetCore, CronScheduler.  

Краткая суть работы заключается в следующем:
1. Получение данных с Яндекс.Погода API и расчет вероятности актированного дня
2. Выкладка результата расчета вероятности в человекочитаемом формате в:  
2.1 Паблик Вконтакте  
2.2. Телеграм бот  
3. Возможность настройки точного времени и количества выкладки результатов.
